# 🏠 Real Estate Backend API

A scalable backend API for managing real estate properties, user authentication, and reviews.

---

## 🚀 Features

- 🔐 JWT Authentication (Login / Register)
- 🏡 Property Management
- ⭐ Review & Rating System
- ☁️ Image Upload (Cloudinary)
- 🧾 Role-based Authorization (Admin / User)
- 📄 API Documentation

---

## 🛠️ Tech Stack

- NestJS
- TypeScript
- MongoDB
- TypeORM
- Cloudinary
- JWT Authentication

## Swagger UI available at:

- http://localhost:3000/api-docs
